Welcome to VimGolf 2021!

Introduction

Vim is known to help us edit text at the speed of thought, however, few of us actually use Vim with such pace and agility. Several users edit files exclusively in Insert mode, or struggle in exiting the editor!

Through this contest, we'll help you develop an appreciation for how fast and powerful Vim really can be!

The idea here is to transform a given input file to a given output file using the lowest number of keystrokes. It'll be the same as vimgolf.com, but with the questions created by us. You'll have few easy, few medium, and few hard questions.

Rules

  1. Contest datetime: 27th Feb 12pm to 28th Feb 12pm
  2. Your commands will be run as the following Vim command: vim -u NONE -Z -n +0 -s solve $f >/dev/null where $f is the name of the input file. You can see the full `vim --version` output here
  3. You can goto the GitHub repo to read how to use the terminal client for this contest
  4. Please join the Discord server for VimGolf 2021: https://discord.gg/MMZR2CBW
  5. Ties are broken by number of problems solved (higher the better), followed by total number of keystrokes (lower the better), followed by date submitted (earlier the better).
