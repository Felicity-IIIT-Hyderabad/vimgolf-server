Welcome to VimGolf 2021!

Introduction

Vim is known to help us edit text at the speed of thought, however, few of us actually use Vim with such pace and agility. Several users edit files exclusively in Insert mode, or struggle in exiting the editor! Through this contest, we'll help you develop an appreciation for how fast and powerful Vim really can be! ૌ The idea here is to transform a given input file to a given output file using the lowest number of keystrokes. It'll be the same as vimgolf.com, but with the questions created by us. You'll have few easy, few medium, and few hard questions.

Rules

  1. Contest datetime: 27th Feb 12pm to 28th Feb 12pm
  2. Your commands will be run as the following Vim command: TODO
  3. For any queries or discussions, use the channel #vimgolf-2021 on Programming Club Discord. Please do not DM the organizers.
  4. Ties are broken by number of problems solved (higher the better), followed by total number of keystrokes (lower the better), followed by date submitted (earlier the better).
